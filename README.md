# Text Editor with Tkinter

Tkinter is a standard Python interface to the Tk GUI toolkit. This is a simple version of text editor and will be added more features in the future.


## Keyboard Shorecuts

|Menu               |Shortcut        |     
|:------------------|:---------------|
|New File           |CTRL+N          |
|Open File          |CTRL+O          |
|Save File          |CTRL+S          |
|Close Current File |CTRL+Q          |
